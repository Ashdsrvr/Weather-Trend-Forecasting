## Overview
This project analyzes global weather data and forecasts future weather trends using Machine Learning and Time Series Forecasting techniques.

## Objective
The objective is to analyze weather patterns, perform exploratory data analysis, detect anomalies, study climate variations, and forecast future weather trends.

## Dataset
Global Weather Repository Dataset from Kaggle.

## Results
- Successfully forecasted future weather trends.
- Compared multiple forecasting models.
- Identified climate and environmental patterns.
- Detected anomalous weather conditions.

## Key Insights
- Temperature varies significantly across regions.
- Air quality is strongly related to weather conditions.
- Climate patterns differ across continents.
- Ensemble models improved forecasting performance.
